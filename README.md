# Oficina de Programação em C

Material de apoio para uma oficina introdutória de programação em C, pensado para turmas iniciantes e organizado em aulas curtas, exemplos comentados e desafios práticos.

## Visão geral

Este repositório reúne materiais para uma oficina de extensão com foco nos primeiros passos em programação. A proposta é apresentar os fundamentos da linguagem C de forma clara, progressiva e acessível, com exemplos simples e contextualizados.

O conteúdo foi organizado para apoiar tanto quem vai ministrar a oficina quanto quem deseja estudar de forma autônoma.

## O que você encontra aqui

- pequena introdução teórica
- exemplos simples em C
- desafios práticos com gabaritos
- materiais de apoio para planejamento e condução da oficina

## Objetivos de aprendizagem

Ao longo da oficina, o material introduz:

- estrutura básica de um programa em C
- entrada e saída com `printf` e `scanf`
- uso de variáveis e operações aritméticas
- condicionais com `if`, `else if` e `else`
- estruturas de repetição com `for`, como tópico opcional
- resolução de problemas simples por meio de lógica de programação

## Público-alvo

Este material foi pensado como o primeiro contato com programação

## Gabaritos

Os gabaritos dos desafios estão em `oficina_programacao_c/gabaritos_desafios`, separados por aula.
