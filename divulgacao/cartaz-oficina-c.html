Oficina de Programação em C

Primeiros passos para entender como programas são criados.

Sobre a oficina

A atividade foi pensada para quem deseja começar a entender como os programas de computador são criados.

Durante os encontros, os participantes vão construir pequenos programas na linguagem C, praticando desde a escrita do código até a execução no computador.

A proposta é apresentar os conceitos iniciais de forma simples, guiada e prática, usando exemplos próximos do dia a dia.

Não é necessário conhecimento prévio em programação.

O que vamos fazer

  • Entender o que é um programa e como ele resolve problemas.
  • Criar pequenos códigos passo a passo, com orientação.
  • Usar o computador para preparar, executar e testar os exemplos.
  • Resolver desafios simples inspirados em situações do cotidiano.
