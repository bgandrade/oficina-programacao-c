# Gabaritos dos desafios

Esta pasta reune as solucoes de referencia para os desafios propostos nas aulas da oficina.

## Organizacao

- `01_aula_01`: gabaritos dos desafios da primeira aula
- `02_aula_02`: gabaritos dos desafios da segunda aula

## Uso sugerido

O ideal e manter os alunos focados primeiro nos arquivos de desafio das aulas. Esta pasta pode ser usada pelo ministrante para:

- conferir uma solucao funcional
- revisar a logica antes da oficina
- apoiar a correcao ou uma explicacao final

As solucoes foram escritas de forma simples e didatica, no mesmo espirito do restante do material.
